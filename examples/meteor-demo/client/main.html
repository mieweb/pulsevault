Each scan opens the chooser in the Pulse app: record a new video, or pick an existing draft to upload to this server.

Auth demo enabled

This server requires a bearer token for every upload and watch request. The QR above embeds it, so the Pulse app handles auth automatically. Tampering with the token in the deeplink should produce a 403 from authorize. To disable, restart the server without DEMO_TOKEN set.
