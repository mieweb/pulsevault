Pair the Pulse app with this local demo server to upload videos over your WiFi network.

How to use
  1. Make sure the Pulse app is on version 1.1.2 Build 3 or higher.
  2. Scan the Save server QR — it adds "Demo Server" to your Pulse destinations.
  3. To upload a specific draft: scan the Upload draft QR and follow the prompt in the app.
  4. (local only) Make sure your phone is on the same WiFi as this computer.
  5. (local only) Open this page from your phone's browser using your laptop's IP address.

Steps marked local only apply when running this demo on your laptop. Skip them if the server is deployed publicly.
